# 🐍 Уроки по Python: от Основ к ООП

Репозиторий с практическими уроками по языку программирования Python, начиная с базовых концепций и заканчивая объектно-ориентированным программированием (ООП). Подходит для новичков и тех, кто хочет систематизировать знания.

# 🎯 Цели

Научить основам Python: переменные, типы данных, операторы, циклы, функции.

Показать принципы ООП: классы, объекты, наследование, инкапсуляция, полиморфизм.

Дать практические задания с решениями.

Создать удобную структуру для изучения и повторения материала.


# 📘 Содержание уроков

## Основы Python
- Переменные и типы данных: int, float, str, bool.
- Условия (if-elif-else) и циклы (for, while).
- Работа с списками, кортежами и словарями.

## Функции
- Определение функций, аргументы, возвращаемые значения.
- Лямбда-функции и рекурсия.
- Модули и импорт библиотек.

## ООП (Объектно-Ориентированное Программирование)
- Классы и объекты.
- Наследование и переопределение методов.
- Инкапсуляция (private/protected атрибуты).
- Полиморфизм и магические методы.
